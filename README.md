# Customer Churn Analysis Dashboard

An end-to-end customer churn analytics project built using **Python, SQL, Streamlit, Pandas, Plotly, and Excel** to analyze telecom customer behavior, churn trends, revenue impact, contract patterns, and customer retention insights.

## Live Demo

https://customer-churn-analysis500.streamlit.app/

## Project Overview

The Customer Churn Analysis Dashboard helps identify why customers leave telecom services and what factors influence churn behavior. The dashboard transforms raw telecom customer data into interactive business insights using data analytics, KPI tracking, SQL analysis, and visual storytelling.

The project focuses on:
- Customer churn analysis
- Revenue insights
- Customer segmentation
- Contract analysis
- Internet service behavior
- Payment method trends
- Customer retention patterns
- Telecom business intelligence

## Features

## Dataset Details
The dataset contains telecom customer information such as:
- Customer ID
- Gender
- Senior citizen status
- Contract type
- Internet service type
- Payment method
- Monthly charges
- Tenure months
- Churn label
  
## Key Insights
- Customers with month-to-month contracts show higher churn probability.
- Higher monthly charges may influence customer churn.
- Internet service type impacts customer retention behavior.
- Payment methods help identify customer billing preferences.
- Long-tenure customers are generally more stable.
- Churn analysis helps telecom companies improve customer retention strategies.
